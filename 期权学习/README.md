南师：不是一个档次的东西，很明显是期权好得多。巴菲特也是期权大师，老头子最著名的期权策略，就是今天谈到的，卖出深度价外看跌期权，赚取权利金，本质就是收租。你们知道2008年次债危机，老巴账面亏了80亿刀萝，但是人家稳得很，因为那其实只是现货。实际情况是在金融危机前，伯克希尔卖出标普500指数15年期看跌期权，执行价约1200点，收了49亿美元权利金。我来告诉你们为啥他一点不慌，有三个原因：  
&emsp;&emsp;其一，49亿刀萝权利金已到手，这是现金；  
&emsp;&emsp;其二，期权到期日是2021-2026年，时间价值归老巴，为什么是5年？因为老巴不是只卖一笔统一的期权，而是分批、分期卖出多笔长期看跌期权，到期日从 2021 年到 2026 年不等；   
&emsp;&emsp;第三，后面标普涨起来，一直涨到今天，所以全部期权作废，他等于白拿了49亿刀萝。这49亿刀萝2008年他就拿到手了，大家听明白了吗？具体逻辑如下：到期时标普500 > 执行价 → 期权作废 → 权利金100%归巴菲特。后来段永平不也是学巴菲特吗？这套玩的六六六。巴菲特也做期货合约，不过他要的期货和各位玩的或者理解的不一样。他做的是长线期货合约，巴菲特不做短期投机，他把期货当作商品企业的延伸来用，用来锁定长期供需价差。逻辑链条如下：  
&emsp;&emsp;期货锁定低价 + 实物交割囤货 → 坐等供需修复。此外巴菲特期货从来不做空，而且不用杠杆，没错，你没看错，全都是自有资金，同时不做短线，另外不做他不熟悉的赛道。虽然巴菲特既做期权，也做期货，但是期权规模要大的多，碾压级的。从目前披露的数据看，伯克希尔的期权规模是期货的五六倍。所以，总结一下，期权是巴菲特的印钞机，期货只是实物资产的辅助工具，是用来赚点外快的。而且大家看一下老巴的期货策略，和贵圈赌狗完全是两码事，所以，我建议你好好学学期权。期货不用放在心上，OK，我讲完了。  
&emsp;&emsp;最后，期权玩的好可以暴富，而且关键是风险可控，但是归根到底，你必须有效管理自己的情绪和仓位，才有机会。

&emsp;&emsp;徒弟：师姐，是上次说的期权ETF吗？  
&emsp;&emsp;园姐：上次说的期权ETF是别人帮你做，你享受就好。 而我今天说的是自己做。 当然这中间是有很多技巧在内的  
&emsp;&emsp;园姐：这张照片里面的很多数字是针对以太的ETF 而言的。最左边是每个星期到期的合同日子。 你可以自己选你的合同有多长时间。 选定时间后， 你又可以选择你要以什么价格来开合同。 美股大部分的股票和ETF都是有期权合同可以选择的，当你买入股票以后，或者是买入ETF以后， 你可以选择你每个星期或者是每个月想要多少钱而作为盈利部分。 当然钱收的越多，那么风险就越大。 这就是你不知道怎么选的话可以把你的股票收走。一般来说，时间越长，收到的权力金就越多。 我这儿说的是一个最简单的期权的操作叫做 selling covered call.  条件是你已经买入了股票或者ETF. 你想持续的持有这个股票，但是又希望从持有这个股票中收到一些固定的收入。那么这个卖出covered call 是最简单，而且形式有效的方法。 还是以上面讲到的Etha 来做个例子。 下面是可以选择的对应不同价格的合同  
&emsp;&emsp;上周五， Etha 关盘的时候， 价格是$33.15。 我就㧥了$39。 这个对应的权力金价格是$1.2 。 那么100股算是一个合同。 每个合同可以收到$120 美金。如果我卖出十个合同， 那么收到$1200 美金。 为什么选$39， 而不是选$33？ 选择$33的合同价， 那么可以收到$3.3/股。 这样一个合同$330， 10个合同就收到$3300。 这个是差不多一个月的收入。 但是风险也大呀，很有可能你的Etha 在$33 就被收走了。 那个时候就是你把ETF卖到了$33+3.3= $36.3，所以选择合同的价格的时候，完全要根据你的心意来做。 你想达到什么样的目标？ 你就可以选什么样的时间和价格。 这个期权是非常灵活机动的。  
&emsp;&emsp;就比如任何人都会买一个币或者买一个股票。 但是你一定会赚钱吗？ 你要达到能够赚钱的目标，这里面的弯弯道道可多了。 这个道理同样适用于买卖期权。 卖出一个合同也是非常简单的事情。 正常智商的人在10分钟内就可以学会。 但是你能保证自己在卖出以后能够达到自己的目的，而且又持续不断地赚取收入吗？ 这个就是每个人的水准不同了。 操作期权的水更深。 想长期盈利的目标还是挺难达到的，这里面有很多认知上面的问题。 随便哪一关出了问题就会给你带来亏损。  
&emsp;&emsp;可以这么说美股sp500中的每一个公司都有期权操作的合同。 目前所有的比特币ETF，eth etf 也都有期权操作的合同。 所以买美股时间长了， 必然会用到期权的操作。 其实前段时间段永平卖出英伟达的长期的covered call. 结果他给自己带来了亏损。 原因在于他卖出的价格太便宜了。 我还是挺喜欢看呱呱蛙博主的文章。 他似乎总是能够找到我找不到的信息。 下面我放二个他的视频。   
&emsp;&emsp;☝️☝️这个视频就是在给段永平今年三月份卖出英伟达的$120元长期的covered call 以后的总结。 当时这个博主在四月就发了一个视频来介绍段永平的这个操作。 但是到七月份的时候， 段永平在卖出的英伟达的$120 c call 上己经损失了二倍以上他当时收到的钱。 也是说， 就算是像投资老手段永平， 他也有操作失误的时间。 而原因在于他对英伟达的定价完全错了。   
&emsp;&emsp;园姐：和投资选股类似。 选择做期权交易的时候，当时的股票价格也非常关键。 一般来说，在市场极为低迷的时候， 是不合适卖出的。 他当时卖出的时候，市场被老特搞得极为恐慌。 他是为了保本， 所以卖出， 收入了$242万美元。 把他的购买价格直接就降下来了。 很多时候， 我们守着几个买入的资产， 如果行情不好。极其单调。 有期权的收入， 会使日子不是那么的难过。 而且有收入进来会给以后的加仓带来了资金。 但是卖出covered call 是有很多弯弯道道在里面的。 用好了绝对是如虎添翼。 所以说投资个课堂，我们有很多东西可以学习。大家不要客气😊🤗分享是一件很愉快的事情。
