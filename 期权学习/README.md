## 期权学习  
### 南山的人：  
南师：学学期权吧，当然归根结底还是应该学什么是价值。    
学生：师傅认为期货和期权，哪个潜在收益风险比更好一些？  

南师：不是一个档次的东西，很明显是期权好得多。巴菲特也是期权大师，老头子最著名的期权策略，就是今天谈到的，卖出深度价外看跌期权，赚取权利金，本质就是收租。你们知道2008年次债危机，老巴账面亏了80亿刀萝，但是人家稳得很，因为那其实只是现货。实际情况是在金融危机前，伯克希尔卖出标普500指数15年期看跌期权，执行价约1200点，收了49亿美元权利金。我来告诉你们为啥他一点不慌，有三个原因：  
&emsp;&emsp;其一，49亿刀萝权利金已到手，这是现金；  
&emsp;&emsp;其二，期权到期日是2021-2026年，时间价值归老巴，为什么是5年？因为老巴不是只卖一笔统一的期权，而是分批、分期卖出多笔长期看跌期权，到期日从 2021 年到 2026 年不等；   
&emsp;&emsp;第三，后面标普涨起来，一直涨到今天，所以全部期权作废，他等于白拿了49亿刀萝。这49亿刀萝2008年他就拿到手了，大家听明白了吗？具体逻辑如下：到期时标普500 > 执行价 → 期权作废 → 权利金100%归巴菲特。后来段永平不也是学巴菲特吗？这套玩的六六六。巴菲特也做期货合约，不过他要的期货和各位玩的或者理解的不一样。他做的是长线期货合约，巴菲特不做短期投机，他把期货当作商品企业的延伸来用，用来锁定长期供需价差。逻辑链条如下：  
&emsp;&emsp;期货锁定低价 + 实物交割囤货 → 坐等供需修复。此外巴菲特期货从来不做空，而且不用杠杆，没错，你没看错，全都是自有资金，同时不做短线，另外不做他不熟悉的赛道。虽然巴菲特既做期权，也做期货，但是期权规模要大的多，碾压级的。从目前披露的数据看，伯克希尔的期权规模是期货的五六倍。所以，总结一下，期权是巴菲特的印钞机，期货只是实物资产的辅助工具，是用来赚点外快的。而且大家看一下老巴的期货策略，和贵圈赌狗完全是两码事，所以，我建议你好好学学期权。期货不用放在心上，OK，我讲完了。  
&emsp;&emsp;最后，期权玩的好可以暴富，而且关键是风险可控，但是归根到底，你必须有效管理自己的情绪和仓位，才有机会。
